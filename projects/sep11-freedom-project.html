SEP 11 Freedom Project

This project was about creating whatever we wanted with javascript as well as other tools to help us. I used Javascript, p5js which is a js visual library for creative coding, and p5lay which is a p5js library that acts as a game engine. By learning how to use these throughout the year I was able to make a game. That game is pinball.

Before and during coding the project I was learning a lot. This project really helped me use and develop many skills as I learned from my challenges. This is a project I am very proud of so I reccomend checking it out as well as my process within my blogs.
